# Wavly 🖐️ — AI-Powered Gesture Interface

Touchless computer control using real-time hand tracking and gesture recognition.

---

## Project Structure

```
wavly/
├── main.py                      # Entry point — starts all threads + tray + keyboard
│
├── core/
│   ├── camera_thread.py         # Thread 1: Webcam → MediaPipe → Gesture Queue
│   ├── action_thread.py         # Thread 2: Gesture Queue → PyAutoGUI actions
│   └── gesture_queue.py         # Thread-safe event bridge
│
├── gestures/
│   ├── classifier.py            # Rule-based + ML gesture classifier
│   ├── landmark_utils.py        # Feature engineering from MediaPipe landmarks
│   └── trainer.py               # Interactive training tool for custom gestures
│
├── ui/
│   ├── tray.py                  # System tray icon — main user entry point
│   ├── settings_window.py       # GUI settings panel (no coding needed)
│   ├── keyboard.py              # Two-hand floating on-screen keyboard
│   └── overlay.py               # Optional: transparent HUD overlay
│
├── config/
│   ├── settings.py              # All tuneable parameters
│   └── gesture_bindings.py      # Gesture → action mappings (auto-managed by UI)
│
├── models/
│   └── gesture_model.pkl        # Trained model (generated by trainer.py)
│
└── requirements.txt
```

## Setup

### 1. Install dependencies

```bash
pip install protobuf==3.20.3 numpy==1.26.4 mediapipe==0.10.14
pip install opencv-python==4.10.0.84 pyautogui==0.9.54 PyQt6==6.7.0 scikit-learn==1.5.1 pillow==10.4.0
```

> Python 3.10 or 3.11 required. Python 3.12 is NOT supported by MediaPipe.

### 2. Run Wavly

```bash
python main.py
```

A hand icon appears in your system tray. Right-click it for options.

### 3. Train your gestures (recommended)

```bash
python gestures/trainer.py
```

Or use the **Settings window → "+ Record gesture"** button — no terminal needed.

## Default Gestures

| Gesture | Action |
|---------|--------|
| ☝️ Index finger only | Move cursor |
| 🤌 Thumb + index pinch | Left click |
| ✌️ Two fingers up, raised | Scroll up |
| ✌️ Two fingers up, lowered | Scroll down |
| ✊ Closed fist | Start drag |
| 🖐️ Open palm | Stop / release drag |
| 🤟 Three fingers up | Toggle on-screen keyboard |

---

## On-Screen Keyboard (Two-Hand Typing)

When the keyboard is open:
- **Left hand** (blue highlight) → hovers over left side keys (Q W E R T / A S D F G / Z X C V B)
- **Right hand** (green highlight) → hovers over right side keys (Y U I O P / H J K L / N M)
- **Pinch** with either hand → types the highlighted key
- **Three fingers again** → closes keyboard

## Adding Custom Gestures (No Coding)

1. Open Settings → click **"+ Record gesture"**
2. Wavly pauses automatically and opens the trainer
3. Follow the prompts, record your gesture
4. Wavly resumes and the new gesture appears in Settings
5. Pick an action from the dropdown → **Save changes**

### Available action types

| Syntax | Example | Effect |
|--------|---------|--------|
| Built-in | `click`, `scroll_up` | Standard actions |
| `show_keyboard` | — | Toggle on-screen keyboard |
| `hotkey:keys` | `hotkey:ctrl+z` | Keyboard shortcut |
| `type:text` | `type:Hello!` | Type a string |
| `run:command` | `run:notepad.exe` | Open an app |

## Configuration

Edit `config/settings.py` to tune:

| Parameter | Default | Effect |
|-----------|---------|--------|
| `hold_frames` | 8 | Frames before gesture fires (anti-jitter) |
| `cursor_smoothing` | 0.35 | 0.1=smooth/laggy, 1.0=raw/fast |
| `scroll_speed` | 3 | PyAutoGUI scroll units per event |
| `show_debug_window` | False | Show OpenCV landmark preview |
| `ml_confidence_threshold` | 0.35 | Min ML confidence before rule fallback |

## Architecture

```
┌─────────────┐    ┌──────────────────────────────────────────────────────┐
│  Webcam     │───▶│  CameraThread                                        │
└─────────────┘    │  OpenCV → MediaPipe (1 or 2 hands) → Classifier      │
                   │  Gesture mode: 1 hand → debounce → GestureQueue      │
                   │  Keyboard mode: 2 hands → hover + pinch → keyboard   │
                   └──────────────────────┬───────────────────────────────┘
                                          │ GestureEvent
                                          ▼
                              ┌─────────────────────┐
                              │   GestureQueue      │
                              └──────────┬──────────┘
                                         │
                   ┌─────────────────────▼──────────────────────────────┐
                   │  ActionThread                                        │
                   │  cursor_move → PyAutoGUI.moveTo (smoothed 60fps)    │
                   │  click/scroll/drag/hotkey/run → PyAutoGUI actions   │
                   │  show_keyboard → signals Qt main thread             │
                   └────────────────────────────────────────────────────┘

                   ┌────────────────────────────────────────────────────┐
                   │  Qt Main Thread                                      │
                   │  WavlyTray — system tray icon                       │
                   │  SettingsWindow — gesture binding GUI               │
                   │  OnScreenKeyboard — two-hand floating keyboard      │
                   └────────────────────────────────────────────────────┘
```

## Troubleshooting

**No tray icon visible**
→ Check the hidden icons arrow `^` in your Windows taskbar
→ Run: `python -c "from PyQt6.QtWidgets import *; app=QApplication([]); from PyQt6.QtWidgets import QSystemTrayIcon; print(QSystemTrayIcon.isSystemTrayAvailable())"`

**ModuleNotFoundError: sklearn / mediapipe**
→ Make sure you're running from inside your `.venv`
→ The trainer always uses the same Python as Wavly automatically

**Cursor is jittery**
→ Increase `cursor_smoothing` in Settings → Sensitivity tab (try lower value like 0.2)
→ Improve lighting — avoid backlit setups

**Gestures fire accidentally**
→ Increase `hold_frames` in Settings → Sensitivity tab (try 12–15)
→ Retrain the model — it will be tuned to your hand

**three_fingers gesture not working**
→ Model needs retraining to include it
→ Run: `python gestures/trainer.py` (or use Settings → Record gesture)

**No hand detected**
→ Set `show_debug_window = True` in `config/settings.py`
→ Check lighting — MediaPipe needs at least ~100 lux

**Keyboard pinch not registering**
→ Bring hand closer to camera
→ Make sure thumb and index tips are clearly touching
